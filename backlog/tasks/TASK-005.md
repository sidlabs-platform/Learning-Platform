## Description

Implement all 10 SQLAlchemy ORM model classes for the domain entities: User, Course, Module, Lesson, QuizQuestion, Enrollment, ProgressRecord, QuizAttempt, ContentGenerationRequest, and ContentGenerationArtifact. Each model must include correct columns, FK relationships with CASCADE delete, UNIQUE constraints, and CHECK constraints as specified in the data-layer LLD.

## Implementation Details

**Files to create/modify:**
- `src/database/models/__init__.py` — exports all 10 model classes
- `src/database/models/user.py` — `User` ORM model
- `src/database/models/course.py` — `Course` ORM model (JSONList for learning_objectives, tags)
- `src/database/models/module.py` — `Module` ORM model (FK to course with CASCADE)
- `src/database/models/lesson.py` — `Lesson` ORM model (FK to module with CASCADE)
- `src/database/models/quiz_question.py` — `QuizQuestion` ORM model (JSONList for options)
- `src/database/models/enrollment.py` — `Enrollment` ORM model (UNIQUE user_id+course_id)
- `src/database/models/progress_record.py` — `ProgressRecord` ORM model (UNIQUE user_id+lesson_id)
- `src/database/models/quiz_attempt.py` — `QuizAttempt` ORM model
- `src/database/models/generation_request.py` — `ContentGenerationRequest` ORM model
- `src/database/models/generation_artifact.py` — `ContentGenerationArtifact` ORM model

**Approach:**
Use SQLAlchemy 2.x `Mapped[]` + `mapped_column()` syntax. All PKs are `String` UUIDs generated by `_uuid()` helper. Apply `__table_args__` for UNIQUE and CHECK constraints. Use `relationship()` with `cascade="all, delete-orphan"` for parent-child relationships (Course→Modules→Lessons, Course→Enrollments). Store `list[str]` fields using the `JSONList` TypeDecorator from TASK-004.

Key constraints:
- `User.role CHECK(role IN ('learner', 'admin'))`
- `Course.status CHECK(status IN ('draft', 'published'))`
- `Course.difficulty CHECK(difficulty IN ('beginner', 'intermediate', 'advanced'))`
- `Enrollment UNIQUE(user_id, course_id)`
- `ProgressRecord UNIQUE(user_id, lesson_id)`
- `Enrollment.status CHECK(status IN ('not_started', 'in_progress', 'completed'))`
